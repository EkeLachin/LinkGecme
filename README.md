# Link Geçme HTML
Basit Link yönlendirme Sitesi.
### Bu kod, yine bir yüklenme animasyonu gösterir ancak bu sefer dairenin içinde bir geri sayım da yapar. Geri sayım, JavaScript kullanılarak setInterval() yöntemiyle oluşturulur ve her saniye sayı 1 azaltılır. Sayfa açıldıktan sonra 5 saniye beklenir ve ardından window.location.href özelliği kullanılarak Google'a yönlendirilir.
